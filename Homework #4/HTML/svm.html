Inline Question 1

It is possible that once in a while a dimension in the gradcheck will not match exactly. What could such a discrepancy be caused by? Is it a reason for concern? What is a simple example in one dimension where a gradient check could fail? How would change the margin affect of the frequency of this happening? Hint: the SVM loss function is not strictly speaking differentiable

$\color{blue}{\textit Your Answer:}$

Our loss function is a RELU, in the sense that we are looking at max(0, x), where x represents the difference in the scores plus some margin delta. If this returns 0, the loss function isn't differentiable (we're at the vertex of the function). This shouldn't be cause for concern, as I think the relative error is still low enough that we likely do not see our algorithm fail.

In our current example, where delta is one, if the difference in scores (defined by the weight times the data x) is equal to 1, we return 0.

Changing the margin would affect how often this value is 0, and we confirm this by looking at the definition of the losses. The closer the scores between the incorrect and correct classes are, the lower delta needs to be in order to create circumstances where max(0, x) returns 0. Likewise, the larger the gap between scores, the larger delta needs to be.
